# Hero FinCorp Analysis

## Overview

This project analyzes loan data to understand default risk, customer behavior, and branch performance.

## Datasets Used

* Customers
* Loans
* Applications
* Defaults
* Transactions
* Branches

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Project Structure

* analysis.py → main code
* data/ → dataset files
* charts/ → output charts
* report.md → detailed analysis

## Key Insights

* Higher interest rates are linked to higher default rates
* Customers with low credit scores are more likely to default
* Some regions have higher default risk
* EMI levels impact repayment behavior

## How to Run

1. Keep all CSV files inside the `data/` folder
2. Run the following command:
   python analysis.py

## Output

Charts are generated and saved in the `charts/` folder.

## Conclusion

This project provides useful insights into loan performance and helps identify key risk factors.
